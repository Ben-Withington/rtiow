## Following [Ray Tracing in One Weekend](https://raytracing.github.io/books/RayTracingInOneWeekend.html#overview)

To run

```g++ -o out/rt -std=c++20 src/*.cpp && out/rt > out/image.ppm```

open ```out/image.ppm``` with a PPM image viewer
